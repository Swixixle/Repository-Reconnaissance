# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

**Do not file public issues for security vulnerabilities.**

To report a security issue, please email: **albearpig@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours.

## Scope

ELI is an early-stage framework specification. It defines schemas and contracts but does not provide production-ready implementations. No security guarantees are made for code in this repository.
