# Trust Summary

**How to verify one claim:**
```
reporecon verify-claim <claim_id> --repo-path <path>
```

**How to audit the dossier:**
```
reporecon audit <dossier.json> --repo-path <path>
```

**What VERIFIED means:**
A VERIFIED claim is supported by cited code at a specified commit, with deterministic evidence and hash.

**Not a security audit:**
PTA does not guarantee security, correctness, or vulnerability-free status. It is not a certification.
