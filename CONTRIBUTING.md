# Contributing to Debrief

Thanks for your interest in contributing. This project is in active development.

## Getting started

See the quickstart in README.md and copy `.env.example` to `.env` before running.

## Development workflow

- `npm run dev` — start the Express + Vite dev server
- `npm run test:unit` — run the Vitest test suite
- `npm run build` — production build

## Pull requests

- Follow Conventional Commits format: `feat:`, `fix:`, `chore:`, `docs:`
- All PRs should target the `develop` branch
- Run `npm run build` and `npm run test:unit` before submitting

## Reporting security issues

See **`docs/SECURITY.md`** — do not open public issues for security vulnerabilities.
